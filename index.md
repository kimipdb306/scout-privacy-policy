# Privacy Policy - Scout OpenClaw

**Last Updated:** February 24, 2026

## Overview

Scout OpenClaw ("Application") is a personal executive assistant powered by Claude AI. This Privacy Policy explains how the Application handles data when accessing Google APIs and services on your behalf.

## Data Collection

The Application collects and processes the following data:

### Google Calendar Data
- Calendar events, titles, descriptions, and dates
- Attendee information (when applicable)
- Calendar metadata

### Data Usage
- To create, read, and manage calendar events as authorized by you
- To provide scheduling and time management assistance
- To integrate calendar data with other productivity features

## Data Storage and Security

- **Local Storage:** Calendar data is processed locally on your device and stored in secure configuration files
- **No Third-Party Sharing:** Calendar data is not shared with third parties except Google (via OAuth2)
- **Encryption:** API credentials and tokens are encrypted in gateway configuration
- **Access Control:** Only authorized users with direct access to your device can access stored data

## User Rights

You can:
- Revoke access at any time via Google Account settings
- Request deletion of cached data
- View what data the Application has accessed

## Google API Permissions

The Application requests the following Google API scopes:
- `https://www.googleapis.com/auth/calendar` — Read and write calendar events

## Data Retention

- Calendar data is not retained beyond the session unless explicitly saved
- API tokens are stored securely for continuous access
- You can delete all data by revoking OAuth2 access in your Google Account

## Changes to This Policy

This Privacy Policy may be updated periodically. Continued use of the Application constitutes acceptance of any changes.
